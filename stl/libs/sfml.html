SFML

craftr/libs/sfml

  • Windows: Downloads the appropriate windows SFML binaries, unless the craftr/libs/sfml:binaryDir option is set.
  • Linux/macOS: Uses pkg-config SFML-all.

Tested on

  • Windows 10
  • Ubuntu 17

Example build script

project "sfml-example"

target "main":
  requires "craftr/libs/sfml"
  cxx.srcs = ['main.cpp']
