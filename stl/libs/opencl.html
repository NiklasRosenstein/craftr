OpenCL

craftr/libs/opencl

Generic build module for using with OpenCL from various vendors.

Options

OptionDescriptionRequiredDefault
:vendorThe name of the OpenCL vendor, can be pkg-config, amd, intel or nvidia.yes (Windows)pkg-config (Linux)
:intelSdkUsed when :vendor=intel, the path to the Intel SDK.noC:\Intel\OpenCL\sdk (Windows)

Note

  • :vendor is a required option on Windows â€“ not setting it causes an error.
  • :vendor=amd is currently not implemented.
  • Using :vendor=nvidia uses the craftr/libs/cuda module. Check this module for options to influence the OpenCL build when using the NVidia OpenCL SDK.
